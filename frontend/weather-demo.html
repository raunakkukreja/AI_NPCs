🎮 How to Test in Game

Press Ctrl+W while playing to open the weather test panel

Or use browser console commands:

// Test different weather conditions
setTestWeather('rain', 15);
setTestWeather('snow', -5);
setTestWeather('storm', 20);
setTestWeather('fog', 12);

// Test temperature effects
setTestTemperature(35); // Very hot
setTestTemperature(-10); // Very cold

// Complete weather data
setTestWeatherData({
  temperature: 30,
  condition: 'storm',
  humidity: 80
});

// Return to real weather
disableWeatherTest();
